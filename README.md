# MAZK Investor Pitch Deck

Pitch deck para inversores construido con React y Vite.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Estructura del Pitch

El pitch incluye las siguientes secciones:

1. **Portada** - Presentación de MAZK
2. **La Historia** - El problema vivido
3. **El Equipo** - Tres fundadores
4. **El Problema** - ¿Quién lo sufre?
5. **Las Fricciones** - Dos opciones terribles
6. **La Plata Sobre la Mesa** - Caso de negocio
7. **La Solución** - Clonación digital
8. **Cómo Funciona** - Setup inicial
9. **La Magia** - Durante la llamada
10. **Por Qué Ahora** - Timing perfecto
11. **Modelo de Negocio** - Unit economics
12. **Tracción** - Dónde estamos hoy
13. **The Ask** - $400K pre-seed
14. **Por Qué Invertir** - Seis razones
15. **Visión** - Cierre
